Calculation flow — shared method for both cases
1
Build calculation table: X₁, X₂, Y, X₁², X₂², X₁Y, X₂Y, X₁X₂
2
Form 3 normal equations using column sums
3
Solve the system of equations for a, b₁, b₂
4
Write regression equation: ŷ = a + b₁X₁ + b₂X₂
5
Compute predicted values and residuals (Y − ŷ)
6
Calculate SSE = Σ(Y − ŷ)² and SST = Σ(Y − ȳ)²
7
R² = 1 − SSE/SST (model fit, expressed as percentage)
8
RMSE = √(SSE/n) (average prediction error)
