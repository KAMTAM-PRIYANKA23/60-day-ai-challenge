# Day 18 – Brain Dump Action Planner

## Objective

Create a reusable Claude Custom Skill that transforms unstructured information into clear summaries, action plans, and learning roadmaps.

The goal was to reduce the effort required to organize notes from different sources and improve productivity through AI-powered workflows.

---

## Tools Used

* Claude AI
* Claude Custom Skills
* Prompt Engineering
* GitHub

---

## Skill Name

```text
brain-dump-action-planner
```

---

## Problem Statement

Information is often scattered across:

* Class notes
* Brainstorming sessions
* Voice memos
* Meeting notes
* Project discussions
* Video transcripts

Manually extracting insights and action items is time-consuming.

This project demonstrates how a reusable AI skill can automate that process.

---

## Skill Description

Convert unstructured notes into organized summaries, action items, risks, blockers, and next steps.

Generate dashboards that help users quickly understand priorities and make decisions without manually reviewing long notes.

---

## Skill Instructions

The custom skill is designed to:

* Summarize key points
* Extract action items
* Identify risks and blockers
* Highlight open questions
* Prioritize tasks
* Generate next steps
* Create interactive dashboards

The skill should work consistently across multiple input formats without requiring prompt re-entry.

---

## Test Inputs

### 1. Multiple Regression Case Study

Prompt used:

```text
Analyze these multiple regression notes.

Generate:

- Executive Summary
- Key Concepts
- Important Formulas
- Action Items
- Areas for Revision
- Open Questions
- Study Plan
- Interactive Dashboard
```

Purpose:

Transform mathematical case study notes into a structured revision plan.

---

### 2. React.js YouTube Transcript

Prompt used:

```text
Analyze this React.js video transcript.

Generate:

- Summary
- Key Topics
- Important Concepts
- Action Items
- Learning Roadmap
- Open Questions
- Next Steps
- Interactive Dashboard
```

Purpose:

Convert long-form video content into actionable learning notes.

---

## Workflow

1. Create the custom skill in Claude.
2. Paste unstructured notes or transcripts.
3. Generate summaries and dashboards.
4. Review action items and blockers.
5. Reuse the skill with different note formats.

---

## Results

The skill successfully transformed raw information into:

* Concise summaries
* Actionable insights
* Prioritized tasks
* Structured learning plans
* Reusable dashboards

---

## Key Learnings

* Reusable workflows are more valuable than one-time prompts.
* Prompt engineering improves output quality.
* AI can simplify information management.
* Structured outputs improve productivity.
* The same workflow can be applied across multiple domains.

---

## Conclusion

This project demonstrated how AI can transform information overload into structured action.

Instead of using AI only for content generation, we can build reusable systems that help us learn, organize, and make better decisions.

> "AI isn't just for generating content—it's for turning information into action."
