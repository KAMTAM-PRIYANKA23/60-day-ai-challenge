# Day 16 – Custom AI Skill for Stock Fundamental Research

## Objective

Create a reusable Claude Custom Skill that automates stock fundamental analysis and generates consistent research reports without repeatedly entering prompts.

---

## Tools Used

* Claude AI
* Claude Custom Skills
* Fundamental Analysis Framework
* GitHub

---

## Skill Name

`stock-fundamental-research`

---

## Problem Statement

Stock research often requires analyzing multiple factors such as:

* Financial performance
* Valuation metrics
* Ownership patterns
* Competitive advantages
* Risk factors

Repeating the same instructions for every stock is inefficient.

Custom Skills solve this problem by enabling reusable AI workflows.

---

## Skill Features

* Company overview generation
* Financial statement analysis
* Valuation comparison
* Ownership trend analysis
* SWOT analysis
* Risk assessment
* Side-by-side stock comparison
* Beginner-friendly explanations
* Visual charts and summaries

## What I Learned

* Custom Skills eliminate repetitive prompting.
* Structured prompts improve consistency.
