What is this?

The Personal AI Playbook is a personal library of reusable AI prompt systems, built for your day-to-day work with Claude — debugging, code review, docs, learning new tech, and architecture decisions. Instead of rewriting the same prompt each time, you build it once as a workflow and reuse it with new details.

An overview: how many workflows you have, which are favorited, and quick links into the builders.
Your saved prompt templates, grouped by category. Each has editable "variables" (like the language or error message) you fill in before copying the final prompt to Claude.
Build a prompt from labeled pieces — Role, Objective, Context, Constraints, Reasoning Strategy, Output Format, Tone, Examples, Quality Checks — so nothing important gets left out. Watch the full prompt assemble live, then copy it or save it as a new workflow.
Wraps a task in instructions that make Claude work autonomously: try, self-check against your criteria, revise, repeat — and stop only when your conditions are met. Useful for things like "keep refactoring until all tests pass."
Everything is stored only in this browser (localStorage). Use Export any time to back it up or move it to another browser; use Import to restore it.
