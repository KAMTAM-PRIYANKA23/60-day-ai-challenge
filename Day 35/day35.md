# 🧩 Prompt Puzzle — Master AI Prompting Through Play

**Prompt Puzzle** is a premium interactive learning game that transforms prompt engineering into a hands-on puzzle experience. Instead of simply reading prompt-writing tips, users actively build, improve, and optimize prompts through engaging drag-and-drop challenges.

Designed as a **single-file HTML application**, it runs entirely offline and requires no backend, APIs, databases, or external assets.

---

# 🎮 Challenge Types

Every game includes **three carefully designed challenge modes**.

## 🧩 1. Build the Prompt

Construct an effective AI prompt using draggable prompt blocks.

Learn:

* Context
* Goal
* Constraints
* Examples
* Formatting
* Tone
* Output requirements

---

## ✂️ 2. Clean the Prompt

Improve an intentionally poor prompt by removing ambiguity, unnecessary instructions, repetition, and weak wording.

---

## ✅ 3. Choose the Best Prompt

Compare multiple prompt versions:

* Weak Prompt
* Optimized Prompt
* Over-Engineered Prompt

Select the version most likely to produce the highest-quality AI output.

>
