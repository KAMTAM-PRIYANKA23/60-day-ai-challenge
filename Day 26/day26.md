Prior Authorization Workflow Simulator
📌 Overview

The Prior Authorization Workflow Simulator is an interactive, gamified web application that visualizes the complete U.S. healthcare Prior Authorization (PA) process. Instead of explaining the workflow through static diagrams, this simulator allows users to experience each step by interacting with draggable patient cases across Patient, Provider, and Payer workflow lanes.

Designed as an educational tool, the application demonstrates how healthcare providers justify medical necessity, collect documentation, submit authorization requests, and respond to payer decisions such as approvals, pends, denials, appeals, and peer-to-peer reviews.

🩺 Patient Scenarios

The simulator includes four realistic healthcare cases with increasing complexity:

🧠 Brain MRI (Low Complexity)
🦵 Elective Knee Replacement Surgery (Medium Complexity)
💉 Specialty Biologic Medication (High Complexity)
🏥 Inpatient Hospital Admission (High / Emergent)

Each scenario follows the same authorization workflow while presenting different clinical challenges.
