About this project

A comprehensive environmental health monitoring dashboard built to visualize and analyze air and water quality data across major Indian cities. The tool combines data analytics with an intuitive UI to make environmental insights accessible to everyone.

Technologies used

HTML5CSS3JavaScript ES6+ Chart.js 4Tabler IconsResponsive Grid Dark ModeCSS Variables

Learning outcomes

  • Interpreting AQI and pollutant thresholds (WHO/CPCB standards)
  • Dashboard design patterns for environmental data
  • Interactive data filtering and comparison techniques
  • Health risk communication through visual design

Future scope

  • Integration with live CPCB / IQAir APIs
  • Historical trend lines and time-series analysis
  • AI-powered pollution forecasting (LSTM models)
  • Mobile app with push alerts for hazardous levels
  • District-level granularity with GIS mapping
