# 📰 Media Integrity Analyzer

An interactive, single-file HTML application designed to teach **media literacy through guided discovery**. Instead of testing prior knowledge, the experience encourages users to observe, analyze, and reflect before revealing expert insights.

Built entirely with **HTML, CSS, and JavaScript**, the application works completely offline without requiring any external libraries, frameworks, APIs, or assets.

---

## ✨ Features

### 🎨 Theme Selection

* Multiple color themes
* Smooth transitions and premium UI

### 🕵️ Challenge 1 – Headline Detective

Learn how headlines can influence perception.

Users will:

* Read a fictional news headline
* Decide whether they would click it
* Identify exaggerated or misleading phrases
* Reveal:

  * Headline Accuracy Score
  * Highlighted misleading sections
  * Detailed explanation
  * Fair and balanced rewritten headline
  * Key media literacy takeaway

### ❤️ Challenge 2 – Emotion Detector

Discover how language influences emotions online.

Users will:

* Analyze a fictional social media post or article excerpt
* Describe their emotional reaction
* Identify influential words or phrases
* Reveal:

  * Intended audience
  * Emotional manipulation strategy
  * Highlighted emotional language
  * Neutral rewritten version
  * Practical takeaway

### 📊 Live Media Integrity Metrics

Real-time indicators update throughout the experience.

* Headline Accuracy
* Source Reliability
* Emotional Manipulation
* Audience Targeting

### 📈 Media Integrity Dashboard

A personalized summary showing:

* Overall Media Integrity Score
* What the user learned
* Biggest media red flag discovered
* Three practical habits for evaluating online content
* Replay option with completely new scenarios

---

## 💡 Learning Approach

Rather than acting like a quiz, the application functions as an interactive lesson where users:

* Observe
* Think critically
* Make predictions
* Compare with expert analysis
* Build lasting media literacy skills

This encourages active learning instead of memorization.
