## 📌 Project Overview

Day 10 was different from every project I have built so far.

Instead of solving an external problem, I focused on creating a professional digital identity that represents my learning journey, technical growth, projects, achievements, and aspirations as an aspiring Software Engineer.

More than a website, this project is a reflection of my commitment to continuous learning and practical application.
