# 🏥 AI Prior Authorization Workflow Simulator

An interactive AI-powered simulation that recreates the healthcare Prior Authorization (PA) process, allowing users to experience how treatment requests move through providers, payers, and approval workflows.

Built as part of my **#60DaysOfClaudeCode** challenge, this project demonstrates how AI can simplify complex healthcare workflows through realistic conversations and decision-based interactions.

## ✨ Features

* 🤖 AI-driven prior authorization workflow
* 🩺 Realistic healthcare approval scenarios
* 💬 Dynamic conversation-based interactions
* 📋 Clinical decision-making simulation
* ✅ Multiple authorization outcomes
* 🔄 Adaptive workflow based on user choices
* 📖 Outcome explanations for every decision
* 🎯 Interactive and educational experience
* 📱 Responsive single-page application
* ⚡ Built with pure HTML, CSS, and JavaScript

## 🚀 How It Works

1. Start a healthcare prior authorization case.
2. Review the patient's clinical information.
3. Make authorization decisions at each stage.
4. Watch the AI respond dynamically based on your selections.
5. Continue through approvals, reviews, or additional documentation requests.
6. Reach the final authorization outcome and understand the reasoning behind every decision.
